Tính năng nổi bật
Trang chủ động: Hiển thị danh sách sản phẩm lấy trực tiếp từ SQL Server.

Giao diện Responsive: Sử dụng Bootstrap 5 để hiển thị tốt trên mọi thiết bị (Mobile, Tablet, Desktop).

Trải nghiệm người dùng (UX):  Menu điều hướng mượt mà với hiệu ứng hover.

Hệ thống Product Overlay (hiện nút Mua nhanh/Xem chi tiết khi di chuột vào ảnh).

Định dạng tiền tệ chuẩn VNĐ (ví dụ: 250,000₫).

Dữ liệu thực tế: Quản lý danh mục (Category), Sản phẩm (Product) và Tài khoản (Account) qua SQL.

🛠 Công nghệ sử dụng
Backend: Java Spring Boot (v3.x), Spring Data JPA.

Frontend: Thymeleaf, HTML5, CSS3, Bootstrap 5, Bootstrap Icons.

Database: Microsoft SQL Server.

Công cụ: IntelliJ IDEA, Maven, Git/GitHub
